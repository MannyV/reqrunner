# Contributing to Req Runner

Thank you for your interest in Req Runner! Here's how you can contribute:

## Getting Started
1. Clone the repository.
2. Load the extension in Chrome (Developer Mode).
3. Set up your Supabase and Gemini API credentials.

## Rules
- Be respectful of others.
- Ensure all code is well-commented.
- Test your changes before submitting a PR.
- Use descriptive commit messages.

## Development
- Use `npm run dev` for a local dev server (if applicable).
- Check the `manifest.json` for extension permissions.
