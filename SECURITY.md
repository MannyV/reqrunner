# Security Policy

## Supported Versions

We currently support the following versions with security updates:
- Version 1.0 (Latest)

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an email to the maintainer directly. We appreciate your help in keeping this project secure.

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
